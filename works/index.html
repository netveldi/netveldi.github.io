Dragon Code

A hobby project meant for players of Ravenloft PotM, a NWN persistent world. It's hosted on github and is a tool that enables players to easily encrypt and decrypt text and use it in the game. The cipher is shifted variant of the combination of the One Time Pad and the Vigenere Cipher.

Rock Paper Scissors Dapp

A small personal project in the works (there's not much to show yet), mainly for me to learn and to show some tangible proficiency with the technologies used. The idea is a rock paper scissors game where the two players bet ethereum and the winner takes all. It is going to be fully decentralized. I'm using the Truffle Suite and the React Box as a base. It uses React, Redux, Jest, Bulma, Web3, Solidity etc. Check it out here. The code is here.
